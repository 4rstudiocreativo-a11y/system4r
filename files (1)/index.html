    📖 Cómo Usar el Sistema

    Guía completa y reglas

    🎮 Sistema de XP

    • • Cada tarea tiene un XP base y tiempo estándar
    • • Tope diario: 250 XP por colaborador
    • • Eficiencia = tiempo_std / tiempo_real
    • • ≥200%: 100% XP ⚡ EXCELENTE
    • • 120-199%: 50% XP 🔥 MUY BUENO
    • • 90-119%: 10% XP ⚡ ACEPTABLE
    • • <90%: 0 XP ⏱ LENTO

    🎁 Sistema de Premios

    • • Solicita premios cuando tengas XP suficiente
    • • Los admins aprueban o rechazan solicitudes
    • • Al aprobar, el XP se descuenta automáticamente
    • • 🟢 Pequeños: 500 XP (canje semanal)
    • • 🔵 Medios: 1,200 XP (2-3 días buenos)
    • • 🟣 Buenos: 2,500 XP (semana fuerte)
    • • 🟡 TOP: 5,000 XP (récords)

    ⏰ Bonos de Asistencia

    • • 08:00: 0 XP bonus
    • • 07:45-07:59: +10 XP
    • • 07:30-07:44: +20 XP
    • • Antes 07:30: +30 XP
    • • Llegada tarde o salida anticipada cancela el bono

    📊 Dashboard y Filtros

    • • Filtra por día, semana o mes
    • • Compara períodos (ej: Enero vs Febrero)
    • • El top performer aparece destacado
    • • Ranking actualizado en tiempo real
